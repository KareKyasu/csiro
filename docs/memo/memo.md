
train.csvの説明

train.csvファイルには、トレーニングサンプルに関する情報が含まれています。各行は1つのトレーニングサンプル（画像）を表し、以下の列が含まれています：

- sample_id: 画像IDとtarget_nameから生成されたIDペア。
- image_path: train/ ディレクトリ内の画像ファイルへのパス。
- Sampling_Date: サンプルが採取された日付。
- State: サンプルが採取されたオーストラリアの州。
- Species: 牧草の種類（バイオマス量の多い順にアンダースコアで区切られています）。
- Pre_GSHH_NDVI: GreenSeekerによる正規化植生指数の測定値。
    正規化植生指数（NDVI）は、植物の健康状態や量の指標となる値です。GreenSeekerは、このNDVIを測定するためのセンサーの商品名です。
    具体的には、センサーが植物に特定の光（赤色光と近赤外光）を当て、その反射率を測定します。
    健康な植物は、光合成のために赤色光を多く吸収し、近赤外光を多く反射します。
    逆に、植物が少ない、あるいは健康でない場合は、その差が小さくなります。
    この反射率の差を計算式（(近赤外光 - 赤色光) / (近赤外光 + 赤色光)）に当てはめて算出されたのがNDVIです。
    このコンペティションでは、NDVIの値が高いほど、その地点の牧草が密生しており、バイオマス量も多い可能性がある、と考えることができます。
- Height_Ave_cm: フォーリングプレート法で測定された牧草地の平均高さ（cm単位）。
- target_name: 予測対象のバイオマス成分の名前。以下の5種類のいずれかです。
  - Dry_Green_g (クローバーを除く乾燥した緑の植物)
  - Dry_Dead_g (乾燥した枯れた物質)
  - Dry_Clover_g (乾燥したクローバー)
  - GDM_g (緑色乾燥物)
  - Dry_Total_g (総乾燥バイオマス)
- target: 上記 target_name に対応する実際のバイオマス量 (グラム単位)。
- その他、Sampling_Date (サンプリング日)、State (サンプリングされたオーストラリアの州)、Species (牧草の種類) などの補助的な情報も含まれており、これらもモデルの入力として利用できます。

注意点: 1つの画像 (image_path) に対して、5つの target_name が存在するため、train.csv 内では1つの画像が5行で表現されます。
