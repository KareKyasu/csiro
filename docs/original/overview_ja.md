# CSIRO - Image2Biomass予測
提供された牧草地画像を用いてバイオマスを予測する

## 概要
画像、グラウンドトゥルース測定値、公開データセットから牧草地バイオマスを予測するモデルを構築します。農家はこれらのモデルを用いて、家畜の放牧時期と放牧方法を決定します。

## 説明
農家は牧草地に入ると、よく「ここには家畜に十分な草があるだろうか？」と自問します。単純なように聞こえますが、答えは決して単純ではありません。牧草地バイオマス、つまり利用可能な飼料の量は、家畜がいつ放牧できるか、いつ休ませる必要があるか、そして牧草地を季節ごとに生産的に維持する方法を左右します。

推定を誤ると、土地は荒廃し、飼料は無駄になり、家畜は苦しみます。正しく行えば、誰もが恩恵を受けられます。動物福祉の向上、より安定した生産、そしてより健全な土壌です。

現在の方法では、この評価は本来の目的以上に困難になっています。昔ながらの「刈り取って計量する」方法は正確ですが、時間がかかり、大規模には不可能です。プレートメーターや静電容量メーターはより迅速に測定できますが、変動する条件下では信頼性が低くなります。リモートセンシングは広範囲のモニタリングを可能にしますが、依然として手作業による検証が必要であり、バイオマスを種ごとに分離することはできません。

このコンテストでは、より環境に優しいソリューションを現場にもたらすことが求められます。画像、地上値、そして公開されているデータセットから牧草地のバイオマスを予測するモデルを構築してください。オーストラリアの牧草地を様々な季節、地域、種の混合にわたって網羅する専門家による注釈付きデータセットとNDVI値を使用して、モデルを強化します。

成功すれば、推定方法の改善にとどまらず、農家がより賢明な放牧選択を行い、研究者が牧草地の健康状態をより正確に追跡できるように支援し、農業をより持続可能で生産性の高いシステムへと推進することができます。

## 評価
スコアリング
モデルのパフォーマンスは、5つの出力次元にわたる$R^{2}$スコアの加重平均を使用して評価されます。最終スコアは次のように計算されます。

```math
\text{最終スコア} = \sum_{i=1}^{5} (w_{i} \times R^{2}_{i})
```
ここで:

項
は次元の決定係数を表します。
使用される重みは
以下のとおりです。
- Dry_Green_g: 0.1
- Dry_Dead_g: 0.1
- nDry_Clover_g: 0.1
- GDM_g: 0.2
- Dry_Total_g: 0.5

## $R^{2}$ の計算
各ターゲットの決定係数 $R^{2}$ は次のとおりです。

```math
R^2 = 1 - \frac{SS_{\text{res}}}{SS_{\text{tot}}}
```

残差平方和$SS_{\text{res}}$

モデルの予測値の総誤差を測定します。

```math
SS_{\text{res}} = \sum_{j} (y_j - \hat{y}_j)^2
```

全平方和 $SS_{\text{tot}}$

データの総分散を測定します。

```math
SS_{\text{tot}} = \sum_{j} (y_j - \bar{y})^2
```

項

- $y_j$: データポイント $j$ のグラウンドトゥルース値
- $\hat{y}_j$: データポイント $j$ のモデル予測値
- $\bar{y}$: すべてのグラウンドトゥルース値の平均

## 提出ファイル
以下の2つの列を含む長形式のCSVを提出してください。

- sample_id: 画像IDとtarget_nameから生成されたIDペア。
- target: 当該のsample_id（float型）に対する予測バイオマス値（グラム）。

有効なターゲット名は、Dry_Green_g、Dry_Dead_g、Dry_Clover_g、GDM_g、Dry_Total_gです。

ファイルには、（画像とターゲット）ペアごとに1行、つまりテストセット内の画像ごとに5行を含める必要があります。
